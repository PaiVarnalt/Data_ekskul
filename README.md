Program ini saya buat buat ngurus data peserta ekskul. 
Di sini admin bisa nambah data peserta, edit data yang salah, hapus data kalau udah ga dipakai, dan semuanya data langsung muncul di halaman index. 
Pas mau edit, datanya otomatis muncul dulu di form jadi kita nggak perlu ngisi ulang dari awal. 
Semua prosesnya pakai PDO biar lebih aman dan biar query-nya nggak gampang error. 
Pokoknya program ini ngebantu banget buat admin pengelolaan data ekskul jadi lebih rapi, cepat, dan gampang dipahami.
